# Welcome!

Welcome to the **Our Community** blog. We are building this platform using a modern hybrid architecture:

- **MDX** for blazing fast, SEO-optimized static content.
- **Supabase** for dynamic interactions like comments, likes, and user profiles.

## Why this approach?

By decoupling our content from our interactive backend, we achieve the best of both worlds. The website loads instantly, yet allows users to log in securely with Google and engage with the community.
